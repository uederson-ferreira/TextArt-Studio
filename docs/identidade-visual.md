## 4. Tipografia

### Filosofia

A UI usa fontes **neutras e funcionais** — o palco pertence às fontes criativas do usuário, não à interface.

### Hierarquia Tipográfica

| Token | Fonte | Peso | Tamanho | Uso |
|- --|---|---|---|---|
| `displayLarge` | Poppins | Bold 700 | 32px | Splash, onboarding |
| `displayMedium` | Poppins | SemiBold 600 | 28px | Títulos de seção |
| `headlineLarge` | Poppins | SemiBold 600 | 24px | Títulos de tela |
| `headlineMedium` | Poppins | Medium 500 | 20px | Subtítulos |
| `titleLarge` | Inter | SemiBold 600 | 18px | Títulos de card |
| `titleMedium` | Inter | Medium 500 | 16px | Labels de seção |
| `bodyLarge` | Inter | Regular 400 | 16px | Texto corrido |
| `bodyMedium` | Inter | Regular 400 | 14px | Texto secundário |
| `bodySmall` | Inter | Regular 400 | 12px | Captions, badges |
| `labelLarge` | Inter | SemiBold 600 | 14px | Botões primários |
| `labelMedium` | Inter | Medium 500 | 12px | Botões secundários |
| `labelSmall` | Inter | Medium 500 | 11px | Tags, chips |

### Espaçamento de Linha

- Headlines: `line-height = font-size × 1.2`
- Body: `line-height = font-size × 1.5`
- Labels: `line-height = font-size × 1.3`

## 9. Animações & Motion

### Princípios

- **Rápido:** Transições de 150–250ms para ações do usuário
- **Natural:** Curvas `easeInOut` para a maioria das animações
- **Expressivo:** Curvas com spring para elementos criativos

### Durações

```bash
instant:    100ms  — feedback tátil (tap)
fast:       150ms  — hover, microinterações
normal:     250ms  — transições de estado
medium:     350ms  — modais, bottom sheets
slow:       500ms  — onboarding, celebrações
```

### Curvas

```dart
standard:   Curves.easeInOut       // navegação geral
decelerate: Curves.easeOut         // elementos entrando
accelerate: Curves.easeIn          // elementos saindo
spring:     Curves.elasticOut      // seleção, confirmação
```

### Microinterações Específicas

| Ação | Animação |
|- --|---|
| Selecionar elemento no canvas | Scale 1.0 → 1.02 com glow violeta |
| Adicionar sticker | Drop + bounce |
| Aplicar fonte | Fade cross entre as duas fontes |
| Desbloqueio premium | Confetti + shimmer dourado |
| Export concluído | Checkmark animado + slide up |
